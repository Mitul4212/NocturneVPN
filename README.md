# NocturneVPN

NocturneVPN is a custom Android VPN client built using Kotlin. It features:

- Fast VPN connection using open protocols
- Server selection by country
- 3D Globe visualization (CesiumJS) of connected IP
- Dark/light theme support
- Clean and responsive UI
